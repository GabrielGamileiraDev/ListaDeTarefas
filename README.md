## 📱 Visão Geral

O **TaskList App** é um aplicativo de lista de tarefas desenvolvido com **React Native** e **Expo**, pensado para ser rápido, intuitivo e funcional tanto no **Android** quanto no **iOS**. Com uma interface limpa e fluída, você organiza sua rotina sem esforço.

<br/>



## 🚀 Como rodar o projeto

### Pré-requisitos

Antes de começar, você vai precisar ter instalado na sua máquina:

- [Node.js](https://nodejs.org/) `>= 18.x`
- [npm](https://www.npmjs.com/) ou [yarn](https://yarnpkg.com/)
- [Expo CLI](https://docs.expo.dev/get-started/installation/)
- [Expo Go](https://expo.dev/go) no seu celular *(opcional)*

<br/>

### 📦 Instalação

```bash
# Clone o repositório
git clone https://github.com/GabrielGamileiraDev/ListaDeTarefas

# Entre na pasta do projeto
cd ListaDeTarefas

# Instale as dependências
npm install
# ou
yarn install
```

<br/>

### ▶️ Executando

```bash
# Inicia o servidor de desenvolvimento
npx expo start
# ou
npm start
```

Após isso, você pode:

- 📱 **Escanear o QR Code** com o app **Expo Go** no seu celular
- 🤖 Pressionar **`a`** para abrir no emulador **Android**
- 🍎 Pressionar **`i`** para abrir no simulador **iOS** *(apenas macOS)*
- 🌐 Pressionar **`w`** para abrir no **navegador**
